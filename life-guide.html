4주 회차 주간 일정

한 회차는 4주 단위로 진행됩니다. Week 1은 오리엔테이션, 4주차는 마무리 평가 및 출국 준비로 구성됩니다.

WEEK 1
오리엔테이션 · 레벨 테스트

OT · 교내 안내 · 레벨 배정 후 정규 수업 시작

WEEK 2
정규 수업 · 버디 활동

오전·오후 수업 + 1:1 Buddy 활동

WEEK 3
정규 수업 · 중간 점검

수업 진행 + 중간 회화 점검

WEEK 4
최종 평가 · 출국 준비

학기말 시험·발표 + 출국 / 다음 회차 등록

현재 회차 시작일 및 상세 시간표는 추후 안내될 예정입니다.

학교 규칙 · 통금 · 외출

안전한 단체 생활을 위해 학교에서 다음 규칙을 운영합니다.

  • 출석 70% 미만 시 ARC / VISA 환불 불가
  • 통금 시간 준수 필수 (자세한 시간은 OT에서 안내)
  • 자유 외출은 학교 정책에 따라 학년·기간별로 제한될 수 있음
  • 음주·흡연·도박은 모두 금지
  • 외부 출입 시 카운터에 출입 등록
환불 정책 · 출석률 70% 미만인 경우 ARC/VISA 비용($200 등) 환불이 불가합니다. 결석은 최소화해주세요.
전체 학칙·벌점 규정은 원본 핸드북에서 확인하실 수 있습니다.

GLOBE 유심 가입

필리핀에서는 통신사 GLOBE 유심을 가장 많이 사용합니다. 입국 후 1주일 이내 가입을 추천합니다.

앱 설치

  • Android: Globe One Super App (Google Play)
  • iPhone: GlobeOne (App Store)

가입 절차

  1. 유심 구매 후 휴대폰에 삽입
  2. Globe One 앱 실행 → Register your SIM 선택
  3. 회원 가입 (이메일·비밀번호 입력)
  4. SIM 등록 양식 작성 — Country: KR South Korea, Type: Foreigner — Tourist
  5. 주소: 학교 주소 입력 (위 입국 안내 참조)
  6. 본인 인증 진행 후 등록 완료
2026 최신 Globe One 앱 화면
가입 단계별 스크린샷을 추후 업데이트 예정입니다.
충전 (Top-up) 방법

충전은 Load Card(편의점 등에서 구매) 또는 앱 결제로 가능합니다.

Load Card로 충전 (단축 코드)

  1. Load Card를 구매하여 뒷면 PIN 코드 확인
  2. 휴대폰에서 *143# 다이얼
  3. 메뉴에서 0 (My Account) 선택
  4. 4 (Load Call Card) 선택
  5. 카드 뒷면 10자리 PIN 입력
  6. 확인 메시지 도착 후 충전 완료
편의점에서 즉시 충전 · 카운터에 본인 번호 + 충전 금액을 말하면 자동 충전 처리되기도 합니다.
데이터 플랜 (Go EXTRA 99)

학생들에게 가성비 좋은 단기 데이터 플랜입니다. 1주일 단위로 갱신합니다.

앱으로 신청

  1. Globe One 앱 실행
  2. 검색창에 GoEXTRA99 입력
  3. Select Payment Method → Charge to own Load
  4. Subscribe 클릭 → 결제 완료
  5. 약 6일간 데이터 사용 가능
2026 최신 데이터 플랜 가격 / 용량
현재 시점 GoEXTRA99의 정확한 데이터·기간을 추후 업데이트 예정입니다.
