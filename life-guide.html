e-Travel 등록 (필수)

필리핀에 입국하는 모든 외국인은 e-Travel 시스템에 사전 등록해야 합니다. 종전의 종이 입국 신고서·세관 신고서가 통합된 전자 신고 시스템으로, 등록 후 발급되는 QR 코드를 입국 심사대에 제시하면 절차가 완료됩니다.

등록 기한 · 출국 72시간 이내에 등록해야 유효합니다. 너무 일찍 등록하면 만료되므로 출국 2~3일 전 사이에 진행하는 것을 권장합니다.

등록 URL: https://etravel.gov.ph

STEP 1 계정 만들기

  1. 홈 화면 중앙의 녹색 Click here to Sign In 버튼을 누릅니다.
  2. 로그인 화면 하단의 Create an account를 선택합니다.
  3. 본인이 자주 사용하는 이메일 주소(Gmail · Naver 등)를 입력하고 Continue.
  4. 입력한 이메일로 발송된 6자리 OTP 인증번호를 입력합니다. 도착까지 최대 3분 소요될 수 있으며, 늦으면 Resend email code로 재발송 가능합니다.
  5. 비밀번호를 설정합니다. 최소 12자 이상이며 대문자 · 소문자 · 숫자 · 특수문자를 모두 포함해야 합니다.
  6. 설정 완료 후 Continue를 눌러 본인 정보 입력 화면으로 이동합니다.
비밀번호 보관 · 동일 계정으로 매 회 재로그인이 가능하니, 비밀번호는 메모장이나 비밀번호 관리 앱에 따로 저장해두시길 권장합니다.

학교 규칙 · 통금 · 외출

안전한 단체 생활을 위해 학교에서 다음 규칙을 운영합니다.

기숙사 생활

  • 학교·기숙사 내 모든 공간에서 음주·흡연 금지
  • 밤 10시 이후 소음은 삼가해주세요
  • 남·여 학생은 서로의 기숙사 출입 불가, 방문객은 1층 로비에서만 접견 가능
  • 기숙사 셀프 스터디 공간은 24시간 운영
  • 귀중품은 개인실 비치된 금고에 보관
  • 기숙사 룸키 분실 시 체크아웃 시 로비에 300 Php 지불

통금 · 외출

  • 기숙사 통금 시간 주중·주말 모두 밤 10시 (이후 외출 제한)
  • 외출 시 기숙사 외출 장부에 이름·목적지·외출 시간 기재, 복귀 후 입실 시간도 기록
  • 늦을 경우 ID 카드 뒷면 연락처로 기숙사 사감 선생님께 연락
  • 주말 외박은 사감 선생님께 신청 후 가능
  • 외박용 투어 예약은 지정 여행사(JTB Cebu, Cebubu)를 통해서만 가능
  • 외박 신청 마감: 매주 목요일 오후 5시
통금 위반 시 · 처음 위반하면 7일간 외출 금지 처분, 28일 이내 다시 위반하면 추가 경고 없이 즉시 퇴교 조치됩니다.

수료 기준

  • 과목별 출석률 70% 이상 + 성적 70% 이상으로 수료해야 수료증 발급
  • 출석률 미달 또는 규정 위반 시 수료증 발급 불가
  • 신청한 연수 기간을 본인 의사로 중단하는 경우 수료증 발급 불가

충전 (Top-up) 방법

충전은 Load Card(편의점 등에서 구매) 또는 앱 결제로 가능합니다.

Load Card로 충전 (단축 코드)

  1. Load Card를 구매하여 뒷면 PIN 코드 확인
  2. 휴대폰에서 *143# 다이얼
  3. 메뉴에서 0 (My Account) 선택
  4. 4 (Load Call Card) 선택
  5. 카드 뒷면 10자리 PIN 입력
